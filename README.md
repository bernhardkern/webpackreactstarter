# webpackreactstarter
this is a simple project that integrates webpack and react. It also uses babel to enable modern es2015 features in all browsers.
<p>
<b>Quickstart:</b><br />
1. npm install<br />
2. npm run watch<br />
3. open browser: http://localhost:3000 <br />
</p>

Technologies used:
React, 
ReactDOM,
webpack,
webpack-dev-server,
babel transpiler
